PYTHON NOTEBOOKS

Kervolutional Neural Networks
Convolutional neural networks are the current state of the art solution for the problem of image classification. A novel approach in the traditional successful CNN is implemented in this project, that uses a projection to higher dimensions using the kernel trick.
The implemenation has better performance by up to 10% in many cases, compared to traditional CNNs for image classification, in terms of classfication error, using the Tiny imagenet library for classification for training.
The Keras library was used for training and evaluating the neural network.

Image captioning with Attention
An image captioning model implemenattion using the Keras library, with an additional Attention mechanism.
The project consists of three main components,an encoder,that takes the image as input and generates a Tensor, a decoder that takes as input the Tensor and an embedding that converts the numerical presentation of the captions into lower dimensions.

C++ PROJECTS

Multiagent MCTS Controller Basic Solution
The implementation of a multiagent system agent for autonomous driving in a lane-free traffic environment.
The agent follows two different approaches. In the first approach, it follows the famous planning Monte Carlo Tree Search(MCTS) algorithm, without learning and without collaborating with other vehicles.
In the second approach, it follows a more sophisticated approach, where a message passing system between agents is supported. This algorithm is also based on the MCTS family of algorithms, with a multiagent setting.
This was my Master Thesis final project.
