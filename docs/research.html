RESEARCH PROJECTS

The following tasks are some of the tasks implemented during my research in the traffic-fluid project.

Multiagent MCTS Controller Basic Solution
The implementation of a multiagent system agent for autonomous driving in a lane-free traffic environment.
The agent follows two different approaches. In the first approach, it follows the famous planning Monte Carlo Tree Search(MCTS) algorithm, without learning and without collaborating with other vehicles.
In the second approach, it follows a more sophisticated approach, where a message passing system between agents is supported. This algorithm is also based on the MCTS family of algorithms, with a multiagent setting.
This was my Master Thesis final project.

SUMO lane-free simulator c++ library tasks
